1.2 Vocabulary and Key Concepts üîë

  • Linear System: A collection of two or more linear equations involving the same set of variables
  • Matrix: A rectangular array of numbers arranged in rows and columns
  • Gaussian Elimination: An algorithm for solving systems of linear equations by transforming the augmented matrix to row-echelon form
  • Augmented Matrix: A matrix representing a linear system with one column appended for constant terms
  • Linear Programming: A method to achieve optimal outcomes in mathematical models with linear constraints
  • Feasible Region: The set of all points satisfying constraints in a linear programming problem
  • Objective Function: The function to be maximized or minimized in optimization problems
  • Residual: The difference between observed and predicted values: \(\text{Residual} = \text{Actual} - \text{Predicted}\)

    Competitive Math Example 1: Pattern Recognition in Tournament Brackets

    Problem: In a single-elimination basketball tournament, the number of games played follows a specific pattern. If a tournament starts with \(n\) teams, how many total games are played?

    Time Challenge: Solve in under 2 minutes using pattern recognition.

    Elegant Solution:

    Instead of calculating round by round, recognize that in any single-elimination tournament:

    • We start with \(n\) teams
    • We need exactly 1 winner
    • Each game eliminates exactly 1 team
    • Therefore, we must eliminate \(n-1\) teams
    • Since each game eliminates exactly 1 team, we need exactly \(n-1\) games

    Answer: \(n-1\) games

    Verification: For \(n = 8\) teams: Round 1 (4 games), Round 2 (2 games), Round 3 (1 game) = 7 games = \(8-1\)

    Critical Thinking Problem 1: Multi-Variable Team Strategy

    A coach is developing a rotation strategy for three players: Alex (\(x\) minutes), Ben (\(y\) minutes), and Carter (\(z\) minutes). The coach has the following constraints:

    • Total playing time cannot exceed 90 minutes
    • Alex must play at least 20 minutes
    • Ben must play at least as many minutes as Carter
    • The "team effectiveness score" is modeled by \(S = 3x + 2y + z\)

    a) Set up the constraint inequalities and identify the feasible region

    b) Find the vertices of the feasible region

    c) Determine the optimal minutes distribution to maximize team effectiveness

    d) Analyze how the solution changes if Alex's minimum requirement increases to 30 minutes

    Solution

    a) Constraints: \(x + y + z \leq 90\), \(x \geq 20\), \(y \geq z\), \(x,y,z \geq 0\)

    b) Vertices: (20,35,35), (20,70,0), (90,0,0), etc.

    c) Optimal solution: Test vertices in \(S = 3x + 2y + z\)

    d) With \(x \geq 30\): Solution shifts, effectiveness may decrease

    Competitive Math Problem 1: Speed Calculation

    Time Limit: 3 minutes

    In a basketball league, teams earn 3 points for a win, 1 point for a tie, and 0 points for a loss. After 20 games, a team has 37 points. If they won 3 more games than they tied, how many games did they lose?

    Solution

    Let w = wins, t = ties, l = losses

    • \(w + t + l = 20\)
    • \(3w + t = 37\)
    • \(w = t + 3\)

    Substituting: \(3(t+3) + t = 37 \implies 4t + 9 = 37 \implies t = 7\)

    Therefore: \(w = 10\), \(l = 3\)

    A basketball player's scoring average after \(n\) games follows the pattern: \(A_n = \frac{20n + 15}{n + 1}\).

    a) What is the player's scoring average after 9 games?

    b) What score must the player achieve in the 10th game to have exactly 25 points per game average?

    c) Prove that the scoring average approaches 20 as \(n \to \infty\)

    Solution

    a) \(A_9 = \frac{20(9) + 15}{9 + 1} = \frac{195}{10} = 19.5\)

    b) Need 10-game average of 25: \(\frac{9(19.5) + x}{10} = 25 \implies x = 74.5\)

    c) \(\lim_{n \to \infty} \frac{20n + 15}{n + 1} = \lim_{n \to \infty} \frac{20 + \frac{15}{n}}{1 + \frac{1}{n}} = 20\)

    Competitive Math Problem 4: Advanced Combinatorics

    Time Limit: 8 minutes

    In a basketball tournament, 16 teams are divided into 4 groups of 4 teams each. Within each group, every team plays every other team exactly once. The top 2 teams from each group advance to a single-elimination playoff.

    a) How many total games are played in the group stage?

    b) How many total games are played in the entire tournament?

    c) If the probability of any team beating any other team is exactly \(\frac{1}{2}\), what is the probability that the #1 seed wins the entire tournament?

    d) Prove your answer using mathematical reasoning

    Solution

    a) Group stage: \(4 \times \binom{4}{2} = 4 \times 6 = 24\) games

    b) Total: 24 (group) + 7 (playoff) = 31 games

    c) Probability = \(\left(\frac{1}{2}\ ight)^3 = \frac{1}{8}\) (must win 3 playoff games)

    d) Proof: Independent games, binomial probability
